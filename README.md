# Go Tournament Manager
Open-source web application for organizing Go tournaments.

## Purpose
This project aims to support Go tournament organizers with a simple and user-friendly web application.

## Target group
The application is primarily designed for tournament organizers. Players should be able to register and view pairings and results.

## Planned MVP features
- tournament creation
- online registration
- participant management
- display of pairings
- display of results
- import/export for pairing software

## Tech stack
- Java
- Spring Boot
- Maven

## Project status
Early prototype / bachelor thesis project.
